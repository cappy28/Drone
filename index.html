Architecture Technique
Stack complète pour une app gamifiée sécurisée, performante et maintenable. Pensée pour un développeur solo avec des coûts d'infrastructure minimaux.
React Native Expo Node.js PostgreSQL Firebase ÉcoleDirecte API Cron Jobs
📱 Frontend Mobile
⚛️
React Native + Expo
Cross-platform iOS/Android depuis une seule codebase. Expo EAS Build pour les binaires. Expo Router pour la navigation file-based.
iOSAndroidOTA Updates
🎨
Animations — Reanimated 3 + Lottie
Barre XP animée avec spring physics, confettis au déblocage d'une pièce, transitions de screens fluides. Lottie pour les animations complexes (confettis, trophées).
60fpsNative Driver
🔔
Notifications Push — Expo Notifications
Notification vendredi 17h via cron job backend → push via Expo Push API → notification locale ou FCM/APNs. Tokens stockés en base.
FCMAPNsCron
💾
Stockage local — MMKV + Zustand
MMKV pour le cache rapide des données (notes, XP). Zustand pour le state global. Pas de données sensibles stockées en clair côté app.
SécuriséRapide
🖥️ Backend API
🟢
Node.js + Fastify (ou NestJS)
API REST avec authentification JWT. Fastify pour les performances. Routes : /auth, /xp, /notes, /shop, /parent. Hébergement Railway ou Render (free tier suffisant).
REST APIJWTRailway
🗄️
PostgreSQL + Prisma ORM
Tables : users (enfant + parents), xp_events (log immuable de chaque gain/perte), weekly_checks, drone_parts, notes_cache. Supabase pour PostgreSQL gratuit + auth intégrée.
PostgreSQLPrismaSupabase
⏰
Cron Jobs — node-cron
Chaque heure : scraping ÉcoleDirecte → mise à jour des notes → calcul XP automatique.
Vendredi 17h : envoi notification push check-in hebdomadaire.
Minuit : calcul des streaks, mise à jour des niveaux.
CronAutomatique
🏫 Intégration ÉcoleDirecte
🔐
Authentification & Sécurité des identifiants
Les credentials ÉcoleDirecte ne sont JAMAIS stockés en clair.
→ Chiffrement AES-256-GCM côté backend avant stockage en base.
→ Clé de chiffrement = variable d'environnement (non versionnée).
→ L'app mobile ne reçoit JAMAIS les credentials, seulement des tokens JWT courts (15 min).
→ Session ÉcoleDirecte stockée en mémoire Redis (TTL 1h).
AES-256Env VarsRedis
📡
Méthode de récupération des notes
Option A (recommandée) : utiliser l'API non-officielle ÉcoleDirecte (endpoints JSON documentés par la communauté).
Librairie npm : ecoledirecte.js ou équivalent.
Option B : Puppeteer headless browser si l'API change (fallback).
Les notes sont normalisées en /20 avant stockage.
API JSONFallback Puppeteer
🛡️ Sécurité & Authentification
👤
Deux comptes distincts
Compte Enfant : accès app mobile, lecture de ses XP, actions auto-déclarées.
Compte Parent : dashboard web React (Vite), validation/refus des actions, ajout de pénalités, lecture totale.
RBACRôles
🔑
Auth — JWT + Refresh Token
Access token 15 min, refresh token 30 jours stocké en HttpOnly cookie. PIN de 6 chiffres pour l'accès parent (2FA optionnel). Aucun mot de passe stocké en clair (bcrypt).
JWTHttpOnlybcrypt
🖼️ Images dynamiques des pièces
📁
Structure des assets
Les images sont dans /assets/drone_parts/ dans le repo React Native.
Chaque pièce a un ID dans la base de données qui correspond au nom du fichier.
Ex : part_motors.png, part_frame.png...
Les images locked sont affichées avec un filtre grayscale + opacité réduite.
Assets locauxDynamic require()
☁️
Alternative GitHub CDN
Si les images doivent être mises à jour sans rebuild de l'app :
→ Héberger sur GitHub Pages (raw.githubusercontent.com/user/repo/main/parts/)
→ L'app récupère les URLs depuis l'API backend.
→ Cache local avec expo-image (LRU cache).
GitHub CDNexpo-image

🤖
Coach IA intégré
Conseil personnalisé basé sur les notes : "Tu as eu 9/20 en Histoire, voici 3 ressources pour t'améliorer." Via Claude API ou OpenAI.

Mode multi-enfants
Dashboard parent gérant plusieurs profils enfants avec classement interne (pour familles nombreuses ou entre amis).
🏆
Défis hebdomadaires
Objectif de la semaine généré automatiquement : "Avoir 2 notes > 15 cette semaine" = bonus +30 XP. Reset chaque lundi.
📈
Prédiction de fin de trimestre
Calcul de la moyenne prévisionnelle par matière basé sur les notes actuelles + projection XP jusqu'à la fin du trimestre.
🎯
Objectifs personnalisés
Les parents peuvent créer des objectifs sur mesure : "Dépasse 16/20 en Sciences = +100 XP bonus" avec date d'échéance.
🌙
Widget iOS/Android
Widget écran d'accueil affichant XP actuel, streak, et prochaine pièce à débloquer. Via Expo Widgets ou react-native-widget-extension.
